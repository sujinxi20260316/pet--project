# 🐾 宠医助手 · 宠物疾病智能问答系统

## 📋 项目简介
这是一个宠物疾病问答系统，可以帮助宠物主人快速了解宠物疾病知识，识别紧急情况。

## ✨ 核心功能
- 🔍 症状匹配分析：输入症状，系统自动匹配相关疾病
- 🚨 紧急情况识别：自动标记需要立即就医的情况
- 📊 疾病知识查询：40+种常见宠物疾病
- 🏥 就医建议指导：提供家庭处理和就医建议

## 🛠️ 技术栈
- 前端：Streamlit
- 后端：Python
- 分词：jieba
- 数据处理：pandas
